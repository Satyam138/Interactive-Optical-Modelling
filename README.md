# Interactive-Optical-Modelling
Here is an interactive too to see the focal plane psf for stars after differacting from different optical system. 
Here You can see the focal plane psf for a star after differacting from different optical systems.
It has majoriity of the telescope aperture built in, which can be accessed by the dropdown menu. 
For the simple obstruction type coronography, you can check the PSF for different obstruction sizes and different numbers of arms and arm width holding the obstruction.
The Wavelength in the unit of lemda/D can be accessed through the slidebars for well-known telescopes such as JWST, GMT, VLT, etc
